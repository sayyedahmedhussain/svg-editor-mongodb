# SVG Editor with MongoDB Integration

A dynamic and interactive SVG Editor that allows users to upload SVG files, edit text elements directly in the browser, and synchronize all changes with MongoDB in real time.

---

## 🚀 Features

- 📂 Upload SVG files from your local computer  
- 🧠 Automatically detect and make SVG text elements editable  
- ✏️ Edit text directly on the SVG canvas  
- 💾 Save edited SVG data to MongoDB  
- 📥 Fetch and load saved SVG content from database  
- 🔄 Real-time synchronization between frontend and MongoDB  
- 🔁 Auto-update SVG when database changes  
- 📱 Fully responsive and user-friendly interface  
- ⬇️ Download edited SVG files  

---

## ⚙️ How It Works

1. Upload an SVG file from your device  
2. The editor detects all editable text elements  
3. Click and modify text directly on the SVG canvas  
4. Save changes to MongoDB  
5. Any updates in MongoDB are automatically reflected in the editor  
6. The frontend stays synced with the database in real time
